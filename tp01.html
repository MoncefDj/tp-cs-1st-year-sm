This simple command displays text on the screen. It's our first step in turning an algorithm into a program.

Variables

A variable is like a box that stores a value. It can be a number, text, etc.

x = 5 y = 2.5 name = "Ali"

Input/Output

input(): reads a value from the user.
print(): displays a value on the screen.

name = input("Enter your name: ") print("Hello", name)
